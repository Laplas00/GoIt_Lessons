    Ваше выпускное приложение сейчас работает в консольном режиме
    и взаимодействует с пользователем посредством команд в консоли.
    Приложение надо развивать и наиболее часто изменяемой частью 
    приложения обычно является пользовательский интерфейс 
    (консоль пока что). Модифицируйте код вашего приложения, 
    чтобы представление информации пользователю 
    (вывод карточек с контактами пользователя, заметками, 
    страничка с информацией о доступных командах) было легко изменить. 
    Для этого надо описать абстрактный базовый класс для 
    пользовательских представлений и конкретные реализации, 
    которые наследуют базовый класс и реализуют консольный интерфейс.
